# LetMeCook
meal prep app that tracks calory goals and recomends recipes to help you reach those goals

## Description
LetMeCook is a smart meal-planning app that tracks calories, logs meals, and recommends recipes tailored to both your tastes and goals. It is ideal for anyone looking to lose weight, gain muscle, or save time in the kitchen by simplifying meal prep and everyday food tracking.

### Features
- Log in and sign up with an email and password!
- Keep track of your calories with a calorie tracker!
- Keep a list of the ingredients in your pantry and get recipes recommended based on what you have on hand!

## Testing (for reviewers)
- Make sure to have Node.js installed (https://nodejs.org/en/download) (v24.14.1 (LTS) for (OS) using Docker with npm)
- Clone the repo (create a folder, right click inside the folder, open terminal, run the command "git clone https://github.com/mess0049/CS3203_GroupA_LetMeCook")
- Run "cd CS3203_GroupA_LetMeCook" to change to the new repo directory
- Run "npm ci" to clean install npm and get all the things we're using in our project (i.e. Firebase and Jest)
- For testing an individual test file, run "npm test -- filename.test.js" (ex: npm test -- auth.test.js) (if this doesn't work, try "npm test filename.test.js" without the --)
- For testing all unit tests, run "npm test"

## Developer Requirements
- Coded in JavaScript through VSCode
